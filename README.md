# Examen-practico-Programacion-basica-Yeray-Perez
en este repositorio subiremos el codigo realizado en el examen con su explicacion en el README





Básicamente, lo que hice fue una clase padre para que todos los productos compartan lo mismo, como el nombre y el presio. Lo más importante es que usé polimorfismo con el método virtual de calcular impuesto, así cada clase hija (la de electrónicos y comida) le mete su propio porciento con el override. También le puse encapsulación con los atributos privados paara que no se hagan pro con los datos desde afuera. Al final en el Main solo cree los objetos y mostré todo para que el profe vea que el calculo de ITBIS sale redondito según el tipo de producto.
